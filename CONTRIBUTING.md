# Using Byte AI - Fork & Build Your Own

Byte AI is **100% open source** under the MIT License. This repository serves as the base version, but we encourage you to **fork it and build your own version**!

## How to Use This Project

### Option 1: Use It As-Is
```bash
git clone https://github.com/RetroGigabyte/Byte_AI.git
cd Byte_AI
./Byte
```

### Option 2: Fork & Customize
We don't accept pull requests to this repository, but you're encouraged to:

1. **Fork the repository**
   - Click the "Fork" button on GitHub
   - This creates your own copy

2. **Clone your fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/Byte_AI.git
   cd Byte_AI
   ```

3. **Make it your own**
   - Add features you want
   - Modify the bot behavior
   - Train on your own data
   - Change the name/branding
   - Customize everything!

4. **Share your version**
   - Publish your fork
   - Show others what you built
   - Build your own community

## Why Fork Instead of Pull Requests?

This approach:
- ✅ Keeps the base version stable and focused
- ✅ Gives you complete freedom to customize
- ✅ Lets you maintain your own direction
- ✅ Prevents feature bloat
- ✅ Encourages community forks with unique features
